# Gnomint icon theme

<img src="overview.png">

## Additions

New icons:

- Fragments
- AnyDesk
- Zoom

Variants from existing icons:

- Firefox Nightly
- Nautilus
- Portfolio

Symlinks to existing icons:

- Gnome apps and other new apps
- New app names (Flathub apps)
